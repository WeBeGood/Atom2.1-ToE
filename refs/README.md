# References

## Purpose
Centralize citations so nodes and papers can share a consistent bibliography.

## Node usage
Nodes may keep local `latex/refs.bib` for node-specific sources.
Prefer adding widely-used references to `refs/global.bib` to avoid duplication.

## Paper usage
Papers can cite from:
- `refs/global.bib`
- plus any paper-local `refs.bib`
