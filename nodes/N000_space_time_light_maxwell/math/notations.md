# N000 Notation + Conventions

## Coordinate system
- Cartesian space: (x,y,z)
- Time parameter: t
- Gradient operator: ∇

## Maxwell sign conventions (vacuum)
- Gauss (electric): ∇·E = 0
- Gauss (magnetic): ∇·B = 0
- Faraday: ∇×E = −∂B/∂t
- Ampère–Maxwell: ∇×B = μ0 ε0 ∂E/∂t

## Orientation
- Cross products and curl use the right-hand rule.
- For plane waves: k̂ points in propagation direction; E ⟂ B ⟂ k̂.

## Symbols
- E: electric field
- B: magnetic flux density
- ε0: vacuum permittivity
- μ0: vacuum permeability
- c: speed of light
