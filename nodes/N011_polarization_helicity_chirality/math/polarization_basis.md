# N011 Math Track — Polarization Bases

## Setup
Take a plane wave propagating along +z:

E(z,t) = Re{ E0 * e^{i(k z - omega t)} }

Transversality requires E0 · ẑ = 0, so E0 lives in the (x,y) plane.

## Linear basis
E0 = Ex x̂ + Ey ŷ

## Circular basis
Define

ê_+ = (x̂ + i ŷ)/sqrt(2)
ê_- = (x̂ - i ŷ)/sqrt(2)

Then
E0 = E_+ ê_+ + E_- ê_-

## Next step
Connect ê_± to explicit time-rotation of the real field E(t) at fixed z,
under an explicit handedness convention.
